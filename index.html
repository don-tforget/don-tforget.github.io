The main difference between our jobs is that we want to work in different departments and locations. Some of us would prefer to work either overseas, with computers (Javascript developer, Programmer) with devices (Application developer) or even with both (Unity developer, Freelancer). We have an all-rounded team because of this.

The main thing that differentiates a Javascript developer from the rest of the ideal jobs is that they’ll be working with main computers (compared to devices) and mainly creating with Java. They won’t have too much interaction with users/customers and will be more in touch with the investors instead. A Javascript developer might have a little in common with a programmer because of the same core concepts behind programming but it’s all dependent on the programming language. Maybe Nicholas and I will work together in the future :”)

The main thing that differentiates a freelancer from the rest of the jobs is predominantly the freedom the job, or jobs, will afford me. As there will not be any specific location I will have to be present in and no fixed job hours i will need to attend, my working experience will essentially be determined by myself only. This will allow me to have a very flexible work life and an ideal work to life balance, which, to that level, is unique to the freelancer job.

The main thing that differentiates an application developer from the rest of the ideal jobs is that it is mainly focus on designing, developing and testing an application. There will be an interaction with the customer as we need feedbacks from them in order to make the application better and satisfy what user needs. The work can actually be done at home, it does not necessary have to be in the office, which means that I will have flexible work time.
